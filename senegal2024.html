Cropland Mapping Project

Google Earth Engine Repository containing all relevant scripts

Presentation

1. Understand crops within the area of interest

  • Overview of region (geography, location, population)
  • Climatology (climate zones, precipitation patterns)
  • Physical characteristics (soils, elevation)
  • Socioeconomic factors (economics, agriculture, food security, staple foods)
  • Crop dynamics (challenges in crop production, crop calendars)

2. Analysis of two existing maps

  • Visual Assessment [relevant script: 1_Worldcover_cropland, 2_visualizing_all_maps]
  • Accuracy Assessment [relevant script: 3_Worldcover_cropland_accuracy]

3. Create cropland for the region

  • Overview of the mapping approach
  • Datasets used (satellite imagery [relevant script: 4_Sentinel2], ground truth data)
  • Models and algorithms employed
  • Visual Assessment [relevant script: 5_classification]
  • Accuracy Assessment [relevant script: 3_Worldcover_cropland_accuracy]
  • Conclusion (key findings, future directions and recommendations)

    function getAOIFromAEZ(zoneList){
    return AEZ.filter(ee.Filter.inList('Zone', ee.List(zoneList)))
    }

    var RiverValleyAndFerlo = getAOIFromAEZ(["Senegal River Valley", "Ferlo"])
    Map.addLayer(RiverValleyAndFerlo, {}, "River Valley and Ferlo")
